Tubular

I chose to demonstrate Tubular

Tubular is a jQuery plugin that lets you set a YouTube video as your page background.

See it in action:

www.seanmccambridge.com/tubular

Why

I chose to work with Tubular because, as a business owner, I plan to use my web development skills to my own advantage. The importance of engaging marketing cannot be overstated in a world filled to its brim with media. When I came across the Tubular website, I was immediately engaged by a moving background. Though the content of the site was secondary to my attention, the video background made it easier to stick around and read through the text. Keeping site visitors around for longer increases opportunity for conversion and that's definitely a goal of mine.

How

After downloading the files from Google Code, I had everything that made the demo site and the means to manifest my own ideas.

This plugin requires jQuery so it is necessary to link to it in your html file.

Include the following code in the head:


<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js" type="text/javascript"></script>

The js folder is really all that is necessary but the css folder contains a reset and the means to sylize one's page similarly to the demo.

Link to the js files in the head:


<script type="text/javascript" charset="utf-8" src="js/jquery.tubular.1.0.js"></script>
<script type="text/javascript" charset="utf-8" src="js/index.js"></script>

I decided to use and modify the css that was provided. If you decide to do the same, note that you'll only need to link screen.css in your html file because it actually imports reset.css.

Link to css in head:


<link href="css/screen.css" rel="stylesheet" type="text/css" />

screen.css contains the following:


@import "reset.css";

So it is unecessary to link reset.css in the head of the html file

In order to include a YouTube video as the background of a web page, Tubular requires just one more thing in the httml file. The very first div to be created will be the container and should have a specific id of "wrapper" so it can be modified by Javascript.

Create a div and with the id "wrapper" as the container for all content:

 
<div id="wrapper">
<!-- Include all page content within this div -->
</div>

At this point, Tubular is all set to run a video as the background of a web page. However, there are just a couple more steps if you want to specify the video that plays.

Find the id of the YouTube video you want to show:

Replace the video id line 2 of index.js:

var options = { videoId: 'e4Is32W-ppk', start: 3 }; //Before
var options = { videoId: 'SuuMjjfDb2Y', start: 0 }; //After

And, for assurances, replace the video id in line 20 of jquery.tubular.1.0.js which establishes defaults:

videoId: 'ZCAnLxRvNNc', //Before
videoId: 'SuuMjjfDb2Y', //After

Where

In order to function, a page running Tubular most be hosted on a web server and will not work locally. I used GitHub to host my demo.
